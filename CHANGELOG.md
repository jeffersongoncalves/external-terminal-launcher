# Changelog

All notable changes to the "External Terminal Launcher" extension are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2026-07-17

### Added
- Terminal-agnostic launcher: Warp, Windows Terminal, WezTerm, Alacritty, kitty.
- Status bar item and command palette entry to open the configured terminal at the workspace root.
- Tab reuse when the terminal supports it.
- Per-terminal executable path override setting.
- PATH-based detection to warn when the selected terminal isn't installed.
